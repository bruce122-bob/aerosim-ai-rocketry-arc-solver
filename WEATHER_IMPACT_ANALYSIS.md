# 天气对火箭Apogee高度的影响分析

## 概述

天气条件对火箭飞行高度有显著影响。本分析基于物理模拟引擎中的实际计算模型，量化不同天气因素对Apogee的影响。

## 主要影响因素

### 1. 温度 (Temperature) 🌡️

**影响机制：**
- **空气密度**：温度越高，空气密度越低（理想气体定律：ρ = P/(R·T)）
- **发动机推力**：温度影响推进剂燃烧速率
  - 温度每升高1°C，推力增加约 **0.4%**
  - 参考温度：20°C
  - 温度范围：-10°C 到 +40°C 时，推力变化可达 ±12%

**定量影响：**
- **低温（0°C）**：空气密度高 → 阻力大，推力低 → **Apogee降低 5-10%**
- **高温（30°C）**：空气密度低 → 阻力小，推力高 → **Apogee增加 5-10%**
- **典型影响范围**：±50-100ft（对于800ft的火箭）

**示例：**
- 基准条件（20°C）：Apogee = 872ft
- 冷天（5°C）：Apogee ≈ 820-840ft（降低 4-6%）
- 热天（35°C）：Apogee ≈ 910-930ft（增加 4-7%）

---

### 2. 大气压力 (Pressure) 📊

**影响机制：**
- **空气密度**：压力越高，空气密度越高（线性关系）
- 标准海平面压力：1013.25 hPa
- 典型变化范围：980-1040 hPa（±3%）

**定量影响：**
- **低气压（980 hPa）**：空气密度低 → 阻力小 → **Apogee增加 2-4%**
- **高气压（1040 hPa）**：空气密度高 → 阻力大 → **Apogee降低 2-4%**
- **典型影响范围**：±20-40ft（对于800ft的火箭）

**示例：**
- 基准条件（1013 hPa）：Apogee = 872ft
- 低气压（990 hPa）：Apogee ≈ 895ft（增加 2.6%）
- 高气压（1030 hPa）：Apogee ≈ 850ft（降低 2.5%）

---

### 3. 湿度 (Humidity) 💧

**影响机制：**
- **空气密度**：水蒸气（H₂O）分子量18，比干空气（平均29）轻
- 湿度越高，空气密度越低
- 使用虚拟温度概念计算：ρ = (P_dry/R·T) + (P_vapor/R_vapor·T)

**定量影响：**
- **低湿度（20%）**：空气密度高 → 阻力大 → **Apogee降低 1-2%**
- **高湿度（80%）**：空气密度低 → 阻力小 → **Apogee增加 1-2%**
- **典型影响范围**：±10-20ft（对于800ft的火箭）

**示例：**
- 基准条件（50% 湿度）：Apogee = 872ft
- 干燥（20% 湿度）：Apogee ≈ 860ft（降低 1.4%）
- 潮湿（80% 湿度）：Apogee ≈ 885ft（增加 1.5%）

---

### 4. 风速 (Wind Speed) 💨

**影响机制：**
- **相对速度**：阻力 = 0.5 × ρ × v_rel² × Cd × A
- **风切变**：风速随高度增加（v(z) = v_ref × (z/z_ref)^0.14）
- **攻角效应**：侧风导致攻角，增加阻力

**定量影响：**
- **无风（0 m/s）**：最小阻力 → **最高Apogee**
- **微风（3 m/s）**：影响较小 → **Apogee降低 1-2%**
- **中风（7 m/s）**：明显影响 → **Apogee降低 3-5%**
- **强风（10+ m/s）**：显著影响 → **Apogee降低 5-10%**

**典型影响范围：**
- 0-3 m/s：±10-20ft
- 3-7 m/s：±30-50ft
- 7-10 m/s：±50-80ft

**示例：**
- 基准条件（0 m/s）：Apogee = 872ft
- 微风（3 m/s）：Apogee ≈ 860ft（降低 1.4%）
- 中风（7 m/s）：Apogee ≈ 840ft（降低 3.7%）
- 强风（10 m/s）：Apogee ≈ 820ft（降低 6.0%）

## 综合影响分析

### 最佳飞行条件（最高Apogee）
- ✅ **高温**（30-35°C）
- ✅ **低气压**（990-1000 hPa）
- ✅ **高湿度**（70-80%）
- ✅ **无风**（0-2 m/s）

**预期增益**：+8-12%（约 +70-100ft）

### 最差飞行条件（最低Apogee）
- ❌ **低温**（0-5°C）
- ❌ **高气压**（1030-1040 hPa）
- ❌ **低湿度**（20-30%）
- ❌ **强风**（8-10 m/s）

**预期损失**：-10-15%（约 -80-130ft）

## 关键结论

1. **温度影响最大**：±5-10%的Apogee变化
2. **风速影响次之**：±3-8%的Apogee变化（取决于风速）
3. **压力影响中等**：±2-4%的Apogee变化
4. **湿度影响最小**：±1-2%的Apogee变化

### 实际飞行建议

1. **选择发射窗口**：
   - 优先选择：温暖、低气压、无风的天气
   - 避免：寒冷、高气压、强风的天气

2. **记录天气数据**：
   - 记录实际飞行时的温度、压力、湿度、风速
   - 用于校准模拟结果

3. **预期精度**：
   - 在相同天气条件下，模拟精度可达 ±5%
   - 天气变化导致的误差：±10-15%

4. **校准建议**：
   - 使用实际飞行数据校准模拟
   - 考虑天气因素对结果的影响
   - 建立天气修正系数

## 技术细节

### 空气密度计算
```
ρ = (P_dry / R·T) + (P_vapor / R_vapor·T)
其中：
- P_dry = P_total - P_vapor
- P_vapor = (RH/100) × P_sat(T)
- P_sat(T) = 611.2 × exp(17.67×T/(T+243.5))
```

### 推力温度修正
```
Thrust(T) = Thrust(20°C) × [1 + 0.004 × (T - 20)]
其中：
- 0.004 = 0.4%/°C 温度系数
- 限制范围：±20%
```

### 风切变模型
```
v(z) = v_ref × (z/z_ref)^α
其中：
- α = 0.14（典型大气条件）
- z_ref = 2m（参考高度）
```
