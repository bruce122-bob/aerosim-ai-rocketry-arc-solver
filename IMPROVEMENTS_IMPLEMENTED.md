## 📝 使用说明

### 使用增强校准
1. 进入 "Analysis" → "Calibration" 标签页
2. 勾选 "Enhanced (Adam)" 复选框
3. 点击 "Start Calibration"
4. 查看改进的校准结果

### 运行不确定性分析
1. 进入 "Analysis" → "Uncertainty" 标签页
2. 设置运行次数（建议1000+）
3. 点击 "Run Analysis"
4. 查看置信区间和风险评估

### 使用自动优化
1. 进入 "Analysis" → "Optimize" 标签页
2. 选择优化模式（角度或参数）
3. 点击优化按钮
4. 查看并应用优化结果

或在 "Simulation" → "Configuration" 页面：
1. 点击 "Find Optimal Angle"
2. 系统自动寻找最佳角度
3. 角度自动应用到配置
