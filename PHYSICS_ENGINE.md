## 1. 🌍 大气模型 (ISA - International Standard Atmosphere)

### 实现依据
- **标准**: 国际民航组织(ICAO)标准大气模型
- **温度递减率**: 0.0065 K/m (对流层)
- **气压公式**: 使用气压高度公式（Barometric Formula）

```
T(h) = T₀ - L·h
P(h) = P₀ · (T/T₀)^(g/(R·L))
ρ(h) = P/(R·T)
```

### 湿度修正
- 湿空气密度 < 干空气密度
- 修正系数: `1 - (humidity/100) × 0.004`

### 声速计算
```
a = √(γ·R·T)
```
其中 γ = 1.4 (空气绝热指数)

## 2. ✈️ 气动阻力模型 (Drag Coefficient)

### 马赫数相关阻力 (Mach-Dependent Drag)

#### 亚音速 (Mach < 0.8)
- 阻力系数基本恒定
- 考虑高度对摩擦阻力的影响

#### 跨音速 (0.8 ≤ Mach < 1.2)
- **阻力激增 (Drag Rise)**: 在 Mach 1.0 附近达到峰值
- 增幅: 1.2倍基础阻力
- 物理原因: 激波形成和气流分离

#### 超音速 (1.2 ≤ Mach < 3.0)
- 阻力逐渐下降
- 使用对数衰减模型

#### 高超音速 (Mach ≥ 3.0)
- 阻力稳定在较高水平

### 高度修正
```
altitude_factor = max(0.85, 1 - h/50000)
```
- 高空空气稀薄 → 摩擦阻力减小

## 3. 🎯 运动方程 (Equations of Motion)

### 2D 弹道飞行模型

状态向量: `[x, y, vₓ, vᵧ]`
- `x`: 水平位移 (m)
- `y`: 高度 (m)
- `vₓ`: 水平速度 (m/s)
- `vᵧ`: 垂直速度 (m/s)

### 力的计算

#### 1. 阻力 (Drag Force)
```
F_drag = 0.5 · ρ · v_rel² · Cd · A
```
- `ρ`: 空气密度 (kg/m³)
- `v_rel`: 相对风速 (m/s)
- `Cd`: 阻力系数 (无量纲)
- `A`: 参考面积 (m²)

**重要**: 阻力方向**总是**与相对速度方向相反

#### 2. 推力 (Thrust Force)
- **导轨阶段**: 推力沿发射角方向
- **自由飞行**: 推力沿速度矢量方向 (重力转向)
- **推力曲线**: 使用线性插值获取实时推力

#### 3. 重力 (Gravity)
```
F_gravity = m · g
```
- 标准重力加速度: 9.80665 m/s²

### 加速度
```
a = F_total / m
```

## 4. 🪂 降落伞物理模型

### 关键参数
- **面积**: `A = π · (D/2)²`
- **阻力系数**: Cd ≈ 1.5 (圆形降落伞的典型值)
- **部署条件**:
  1. 越过顶点 (vᵧ < -1.0 m/s)
  2. 发动机熄火
  3. 高度 > 5m (安全裕度)

### 终端速度 (Terminal Velocity)
```
v_terminal = √(2·m·g / (ρ·Cd·A))
```

典型值: 5-8 m/s (安全着陆速度)

## 5. 🌬️ 风力影响

### 风速矢量分解
```
wind_vₓ = -wind_speed · cos(wind_direction)
wind_vᵧ = 0  (假设水平风)
```

### 相对速度
```
v_rel = v_rocket - v_wind
```

**物理意义**: 火箭受到的阻力取决于相对于空气的速度，而非相对地面的速度。

## 6. ⚙️ 推进剂质量模型

### 线性燃烧假设
```
m(t) = m_dry + m_motor_total - (t/t_burn) · m_propellant
```

### 质量守恒
- 发动机熄火后: `m = m_dry + m_motor_case`
- 确保质量不低于干质量

## 7. 🔢 数值积分方法

### RK4 (4阶龙格-库塔法)

**精度**: O(Δt⁴)

**为什么选择RK4?**
- 比欧拉法精度高1000倍以上
- 数值稳定性好
- 计算开销适中
- 被OpenRocket、NASA等机构广泛使用

### 时间步长
- Δt = 0.02 s (50 Hz)
- 足够捕捉快速动态变化
- 保持实时计算性能
