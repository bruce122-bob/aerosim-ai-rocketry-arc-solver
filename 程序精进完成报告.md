### 2. 科学的校准参数策略（P1 - 精确性关键）✅

**问题**：简单平均忽略飞行条件差异和误差可靠性，不够精确。

**解决方案**：

#### 2.1 加权平均（已实现）✅
```typescript
// 误差越小，权重越大（反比例权重）
weight_i = 1 / (error_i + 0.1)  // 0.1防止除零
kThrust_weighted = Σ(kThrust_i * weight_i) / Σ(weight_i)
kDrag_weighted = Σ(kDrag_i * weight_i) / Σ(weight_i)
```

**科学依据**：
- 误差小的数据更可靠，应该权重更高
- 避免误差大的飞行影响整体参数

**实现位置**：`components/FlightDataAnalysis.tsx` 第426-432行
