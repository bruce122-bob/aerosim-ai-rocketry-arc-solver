## 📚 使用说明

### 快速开始

1. **安装依赖**:
```bash
npm install
```

2. **运行开发服务器**:
```bash
npm run dev
```

3. **访问**:
```
http://localhost:3001
```

### 核心API

#### 基础模拟
```typescript
import { runSimulation } from './services/physics6dof';

const result = runSimulation(rocket, env, 90, 1.0);
```

#### 蒙特卡洛分析
```typescript
import { runMonteCarloAnalysis } from './services/monteCarlo';

const mcResult = runMonteCarloAnalysis(
  rocket, env, 90, 1.0, 
  DEFAULT_UNCERTAINTY, 
  1000
);
```

#### 风险评估
```typescript
import { assessRisk } from './services/monteCarlo';

const risk = assessRisk(mcResult, 50, 500, 10);
```
