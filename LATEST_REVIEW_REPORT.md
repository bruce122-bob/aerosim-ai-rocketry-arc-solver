```653:671:services/physics6dofStable.ts
  // 3.1 攻角效应对阻力的影响（风引起的附加阻力）
  // 真实火箭在侧风下会产生攻角，导致额外阻力
  // 对于稳定火箭，攻角通常很小，但仍有影响
  if (vRelMag > 1.0 && env.windSpeed > 0.5) {
    // 计算攻角：火箭轴向与相对速度的夹角
    const vRelHat = vec3.normalize(vRel);
    const cosAlpha = vec3.dot(state.axis, vRelHat);
    const alpha = Math.acos(Math.max(-1, Math.min(1, cosAlpha)));
    
    // 限幅：稳定火箭的攻角通常很小（< 7°）
    const alphaClamped = Math.min(alpha, 0.12); // 0.12 rad ≈ 6.9°
    
    // 攻角引起的附加阻力系数
    // Cd_alpha ≈ 1.5-2.0 对于典型火箭，使用适中值
    const Cd_alpha = 1.5;  // 与 physics6dof.ts 一致
    const Cd_AoA = Cd_alpha * alphaClamped * alphaClamped;  // 小角度时近似二次
    
    cd += Cd_AoA;
  }
```

## 🎓 技术参考

### 攻角效应实现

**公式：**
```
alpha = acos(dot(axis, vRelHat))
alphaClamped = min(alpha, 0.12 rad)  // 限幅6.9°
Cd_AoA = 1.5 * alphaClamped²
Cd_total = Cd_base + Cd_AoA
```

**物理依据：**
- 侧风导致火箭产生攻角
- 即使稳定火箭也有小攻角
- 攻角阻力与攻角的平方成正比（小角度近似）

**系数：**
- Cd_alpha = 1.5（典型值范围1.0-2.0）
- 保守估计，适用于稳定火箭
