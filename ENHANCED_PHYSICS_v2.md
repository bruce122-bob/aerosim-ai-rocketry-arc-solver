### 1. **湿度对空气密度的影响** 🌧️
```typescript
// 水汽比干空气轻约38%，影响空气密度
ρ = (P_dry / R·T) + (P_vapor / R_vapor·T) × 0.622
```
- **饱和蒸汽压** (Magnus公式)
- **实际蒸汽压** = 湿度% × 饱和蒸汽压
- **虚拟温度**修正
- **影响**：高湿度(80%+) → 空气密度降低 ~1% → 阻力减小 → 高度略增

### 2. **温度对发动机推力的影响** 🌡️
```typescript
// 低温降低推进剂性能，高温增加过压风险
推力修正系数 = 1.0 + (T - 20°C) × 修正因子
```

#### 温度修正表：
| 温度范围 | 推力变化 | 建议 |
|---------|---------|------|
| < 0°C | -0.3%/°C | ⚠️ 冷天降低~6% |
| 0-20°C | -0.3%/°C | 推力略降 |
| 20°C | 基准 | ✅ 标准条件 |
| 20-35°C | +0.1%/°C | 推力略增 |
| > 35°C | +0.1%/°C | ⚠️ 过压风险 |

**实例**：
- `-10°C`：推力降低 **9%** (如果标准推力50N → 实际45.5N)
- `30°C`：推力增加 **1%** (50N → 50.5N)

### 3. **风切变效应（Wind Shear）** 💨
```typescript
// 风速随高度增加（幂律模型）
风速(h) = 风速(参考) × (h / h_参考)^α
```

#### 风切变指数 α：
- **开阔地形**：0.14 (NASA标准)
- **城市/树林**：0.25-0.30
- **水面**：0.10

**实例**：
- 地面风速：5 m/s
- 100m高度：5 × (100/2)^0.14 = **7.2 m/s** (+44%)
- 300m高度：**8.9 m/s** (+78%)

**影响**：
- 增加高空阻力
- 轨迹偏移加剧
- Apogee 降低 5-10%

### 4. **改进的跨音速/超音速阻力模型** 🔊

#### 马赫数区间：
| 马赫数 | 流动状态 | Cd 倍数 | 物理现象 |
|--------|---------|---------|----------|
| M < 0.3 | 不可压缩 | 1.0× | 正常阻力 |
| 0.3 - 0.8 | 亚音速可压缩 | 1.0-1.3× | Prandtl-Glauert修正 |
| 0.8 - 1.0 | 跨音速 | 1.0-5.0× | **音障，激波阻力** |
| 1.0 - 1.2 | 刚超音速 | 2.0-3.0× | 激波稳定 |
| 1.2 - 2.0 | 超音速 | 1.8-2.2× | 阻力逐渐降低 |
| M > 2.0 | 高超音速 | 1.8× | 阻力平台 |

**关键区域：M = 0.95-1.05（音障）**
- 阻力增加 **300-500%**
- 需要极大推力突破
- 业余火箭很少达到此区域

### 5. **发射安全检查系统** 🛡️

#### 自动评估项目：

##### A. 风速检查
```
✅ < 5 m/s   : 风速良好
✓  5-10 m/s  : 风速可接受
⚠️ 10-15 m/s : 风速过大，谨慎发射
🚨 > 15 m/s  : 危险，强烈建议取消
```

##### B. 温度检查
```
⚠️ < 0°C   : 温度过低，推力降低
✅ 0-35°C  : 温度适宜
⚠️ > 35°C  : 温度过高，过压风险
```

##### C. 推重比（Thrust-to-Weight Ratio）
```
⚠️ T/W < 3.0  : 推重比过低，难以稳定离轨
✅ 3.0-15.0   : 推重比良好
⚠️ T/W > 15.0 : 推重比过高，加速度可能损坏火箭
```

**推重比公式**：
```
T/W = 初始推力 / (发射质量 × 重力加速度)
```

**推荐值**：
- **最低**：5.0 (稳定飞行)
- **理想**：8.0-12.0 (业余火箭)
- **竞赛**：10.0-15.0 (NASA SLI)

##### D. 导轨长度检查
```
⚠️ < 0.8m : 导轨过短
✅ 0.8-2.0m : 导轨长度合适
✓  > 2.0m  : 导轨较长，稳定性极佳
```

##### E. 离轨速度检查（实时）
```
🚨 < 12 m/s : 离轨速度过低，火箭不稳定
✅ ≥ 12 m/s : 离轨速度足够，火箭稳定
```

**最小离轨速度**：
- **理论最低**：12 m/s (~40 ft/s)
- **推荐**：15 m/s (~50 ft/s)
- **理想**：20 m/s (~65 ft/s)
