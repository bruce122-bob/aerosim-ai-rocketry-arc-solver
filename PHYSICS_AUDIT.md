## ✅ 当前正确的物理模型

### 1. **数值积分** - Runge-Kutta 4 (RK4)
- ✅ **状态**：正确
- **依据**：四阶RK4是求解常微分方程的标准高精度方法
- **误差**：O(Δt⁴) - 非常小
- **时间步长**：dt = 0.02s (50Hz) - 合理

### 2. **大气模型** - ISA (International Standard Atmosphere)
- ✅ **状态**：正确
- **公式**：
  - T(h) = T₀ - L·h (温度递减)
  - P(h) = P₀·(1 - L·h/T₀)^(g/RL) (气压随高度)
  - ρ = P/(R·T) (理想气体状态方程)
- **用户输入**：使用用户设置的T₀, P₀ (正确)

### 3. **湿度修正**
- ✅ **状态**：正确
- **公式**：Magnus公式计算饱和蒸汽压
- **物理依据**：水汽分子量(18) < 干空气(29) → 湿空气密度更低

### 4. **风切变** (Wind Shear)
- ✅ **状态**：正确
- **公式**：v(h) = v_ref × (h/h_ref)^α
- **指数α**：0.14 (开阔地形标准值，来自实测数据)

### 5. **重力**
- ✅ **状态**：正确
- **值**：g = 9.80665 m/s² (标准重力加速度)

### 6. **推力**
- ✅ **状态**：正确 (插值)
- ⚠️ **数据源**：推力曲线需要真实测试数据

### 7. **温度对推力的影响**
- ✅ **状态**：正确

## ⚠️ 需要重新审视的模型

### 8. **风引起的阻力增加** ❓

#### 当前模型 (可能过度简化)：
```typescript
Cd_effective = Cd_base × (1 + windSpeed × 0.015)
```

#### 物理原理分析：
侧风会导致火箭产生**攻角 (Angle of Attack, α)**：

```
α ≈ arctan(v_wind / v_rocket)
```

攻角导致的额外阻力：
```
Cd_AoA = Cd_base + Cd_α × α²
```

其中 Cd_α 取决于火箭几何形状。

**问题**：我当前的线性模型过于简化！
- ✅ 正确：风确实增加阻力
- ❌ 错误：不应该是简单的线性关系
- ❌ 错误：应该考虑火箭速度（风速/火箭速度的比值）

#### 正确的物理模型：
```typescript
// 1. 计算相对风速的攻角
α = atan(v_wind / v_rocket)

// 2. 攻角引起的额外阻力（小角度近似）
Cd_extra = k × α²   // k取决于火箭形状

// 3. 总阻力系数
Cd_total = Cd_base + Cd_extra
```

**这才是基于物理的模型！**

### 9. **马赫数相关阻力** - 跨音速/超音速

#### 当前模型：
```typescript
M < 0.8:  Cd = Cd_base
0.8-1.0:  Cd = Cd_base × (1 + 4×((M-0.8)/0.2)²)
1.0-1.2:  Cd = Cd_base × (2 + (M-1))
> 1.2:    Cd = Cd_base × 1.8
```

#### 物理依据：
- ✅ Prandtl-Glauert修正 (亚音速)
- ✅ 跨音速阻力峰值
- ✅ 超音速激波阻力
- ⚠️ **但系数需要验证是否适用于火箭**

**文献参考**：
- NASA TN D-629: "Drag of Axisymmetric Blunt Bodies at Transonic and Supersonic Speeds"
- 火箭形状的Cd在M=1.0附近增加约150-200%

**评估**：当前模型大致正确，但可能对业余火箭（很少达到M>0.8）影响不大
