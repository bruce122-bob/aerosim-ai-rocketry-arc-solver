### 3. **组件添加面板** ⭐⭐

#### 新增功能：
- ✅ **面板标题**：显示 "Add new component"
- ✅ **添加提示**：显示当前将添加到哪个组件
- ✅ **组件提示卡片**：💡 Component Tips

- ✅ **快速统计卡片**：📊 Rocket Stats
  - 总组件数
  - 总质量
  - 稳定性
  - 长度
  - 最大直径
- ✅ **悬停效果**：按钮悬停时蓝色边框高亮

#### 组件分类：
```
1. Assembly Components (组装组件)
   - Stage (级)
   - Boosters (助推器)
   - Pods (吊舱)

2. Body Components and Fin Sets (机体组件和翼组)
   - Nose Cone (鼻锥)
   - Body Tube (机体管)
   - Transition (过渡段)
   - Trapezoidal Fins (梯形翼)

3. Internal Components (内部组件)
   - Inner Tube (内管)
   - Centering Ring (定心环)
   - Engine Block (发动机挡块)
   - Launch Lug (导轨)

4. Recovery (回收系统)
   - Parachute (降落伞)
   - Shock Cord (缓冲绳)

5. Mass Objects (质量对象)
   - Mass Component (质量块)
```

## 🎯 使用指南

### 基本操作

#### 选择组件
```
方法1: 点击组件树中的组件
方法2: 点击右侧示意图中的组件（如已实现）
```

#### 编辑组件
```
方法1: 双击组件
方法2: 选中后点击 "Edit" 按钮
方法3: 右键菜单 → Edit
方法4: 悬停组件 → 点击编辑图标
```

#### 移动组件
```
方法1: 选中后点击 "Move up/down" 按钮
方法2: 右键菜单 → Move up/down
```

#### 复制组件
```
方法1: 选中后点击 "Duplicate" 按钮
方法2: 右键菜单 → Duplicate
方法3: 悬停组件 → 点击复制图标
```

#### 删除组件
```
方法1: 选中后按 Delete 键
方法2: 选中后点击 "Delete" 按钮
方法3: 右键菜单 → Delete
```

#### 添加组件
```
1. 选中父组件（Stage 或 Body Tube）
2. 右侧面板中点击要添加的组件类型
3. 新组件将被添加到选中组件下
```

#### 快捷操作图标
```typescript
<div className="opacity-0 group-hover:opacity-100 transition-opacity">
  <button onClick={edit}>Edit</button>
  <button onClick={duplicate}>Copy</button>
</div>
```

#### 统计计算
```typescript
// 组件总数
const countComponents = (comps: RocketComponent[]) => {
  let count = 0;
  const traverse = (c: RocketComponent[]) => {
    c.forEach(comp => { count++; traverse(comp.subComponents); });
  };
  traverse(comps);
  return count;
};
```
