```418:432:components/FlightDataAnalysis.tsx
                // Calculate statistics with weighted average (more precise)
                // Weighted average: error越小，权重越大（反比例权重）
                // weight_i = 1 / (error_i + 0.1)  // 0.1防止除零
                // This ensures that more accurate calibration results have higher influence
                const kThrustValues = results.map(r => r.kThrust);
                const kDragValues = results.map(r => r.kDrag);
                const errors = results.map(r => r.error);

                // Calculate weights (inverse of error, with small offset to prevent division by zero)
                const weights = errors.map(error => 1 / (error + 0.1));
                const totalWeight = weights.reduce((a, b) => a + b, 0);

                // Weighted average for kThrust and kDrag
                const avgKThrust = kThrustValues.reduce((sum, val, idx) => sum + val * weights[idx], 0) / totalWeight;
                const avgKDrag = kDragValues.reduce((sum, val, idx) => sum + val * weights[idx], 0) / totalWeight;
```

## ⚠️ **严重问题发现：enhancedCalibration.ts 参数传递错误**

### 问题描述

`enhancedCalibration.ts` 中的 `calculateRMSE` 函数（第55行）调用 `runSimulation` 时**没有传递校准参数**！

```52:56:services/enhancedCalibration.ts
    const calculateRMSE = async (kThrust: number, kDrag: number): Promise<number> => {
        // 6DOF系统不再需要PhysicsConfig，直接使用默认参数
        // 注意：kThrust和kDrag参数在6DOF系统中需要通过其他方式应用
        const simResult = await runSimulation(rocket, env, launchAngleDeg, railLength);
        const simulatedApogee = simResult.apogee; // in meters
```

**影响**：
- Adam优化器无法正确测试不同的 kThrust 和 kDrag 值
- 所有优化迭代都会使用默认参数（kThrust=1.0, kDrag=1.0）
- 优化过程实际上无效
- 最终返回的优化参数是随机的，不是真实优化的结果

### 修复要求

需要在调用 `runSimulation` 前，将 kThrust 和 kDrag 设置到 `rocket.simulationSettings`：

```typescript
const rocketWithParams: RocketConfig = {
    ...rocket,
    simulationSettings: {
        ...rocket.simulationSettings,
        kThrust,
        kDrag
    }
};
const simResult = await runSimulation(rocketWithParams, env, launchAngleDeg, railLength);
```
